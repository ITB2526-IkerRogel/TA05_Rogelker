Desenvolupament Web FullStack

React & Node.js

Aquest projecte consisteix en la creació d'una plataforma web completa dissenyada per a la gestió d'usuaris i continguts en temps real. S'ha utilitzat React per a un frontend dinàmic i reactiu, connectat a una API REST desenvolupada amb Node.js i Express.

La base de dades utilitzada és MongoDB, permetent una escalabilitat horitzontal. El projecte inclou autenticació segura (JWT), càrrega d'imatges i un panell d'administració.

Ciberseguretat: Pentesting

Kali Linux & Wireshark

Auditoria de seguretat exhaustiva realitzada en un entorn corporatiu simulat. L'objectiu principal va ser identificar i explotar vulnerabilitats crítiques per posteriorment corregir-les (Hacking Ètic).

Es van utilitzar eines com Nmap per a l'escaneig de xarxes, Metasploit per a l'explotació i Wireshark per a l'anàlisi de trànsit. L'informe final detalla els vectors d'atac i les mesures de mitigació recomanades.
