About the Project

This project was born from the need to create a smooth and secure online shopping experience. I developed an architecture based on (simulated) microservices using the MERN stack, prioritizing scalability and loading speed.

The system allows full inventory management by administrators, while users enjoy a simplified checkout process with integrated payment gateway via Stripe API.

Key Features

Optimal Performance

Lazy-loading for images and database query optimization.

JWT Security

Robust authentication with access and refresh tokens to protect sessions.

Responsive Design

"Mobile First" design adapted to any device.

Admin Dashboard

Control panel with real-time sales charts.
