Cybersecurity Ops

Corporate Network
Pentesting Audit

Audit Objective

The main objective of this project was to evaluate the resilience of the network infrastructure of a simulated company "TechCorp" against external and internal attacks. A Black Box penetration test was performed following the PTES methodology.

During the process, multiple critical attack vectors were identified, including outdated services and default configurations on production servers.

Executed Phases

Recon & Scanning

Passive information gathering and active port scanning with Nmap.

Exploitation

Use of Metasploit to exploit known CVE vulnerabilities.

Privilege Escalation

Privilege escalation on Linux systems using Kernel exploits.

Reporting

Technical and executive documentation with CVSS scoring.

Tools

Kali Wireshark Bash Python
